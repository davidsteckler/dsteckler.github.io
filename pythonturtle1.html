Step 1: Import the Turtle Module

The Turtle module comes pre-installed with Python, so you don't need to install anything extra. To use it, you first need to import it.

import turtle

Step 2: Create a Turtle Object

Next, create a Turtle object. This object will be your "pen" that draws on the screen.

import turtle

# Create a screen
screen = turtle.Screen()

# Create a turtle named "pen"
pen = turtle.Turtle()

Step 3: Move the Turtle Forward

Let's make the turtle move forward by 100 units. This will draw a straight line.

pen.forward(100)

Step 4: Turn the Turtle Right

To draw a square, the turtle needs to turn 90 degrees to the right after each side.

pen.right(90)

Step 5: Repeat Moving and Turning

To complete the square, repeat the move and turn steps three more times.

import turtle

screen = turtle.Screen()
pen = turtle.Turtle()

# Draw first side
pen.forward(100)
pen.right(90)

# Draw second side
pen.forward(100)
pen.right(90)

# Draw third side
pen.forward(100)
pen.right(90)

# Draw fourth side
pen.forward(100)
pen.right(90)

Step 6: Add a Loop to Simplify

Instead of writing the same code four times, use a loop to make the code cleaner.

for _ in range(4):
    pen.forward(100)
    pen.right(90)

Step 7: Keep the Window Open

After drawing, you want the window to stay open until you click on it.

screen.exitonclick()

Complete Code

Here's the complete code for Challenge 1:

import turtle

# Set up the screen
screen = turtle.Screen()
screen.title("Challenge 1: Draw a Square")

# Create a turtle named "pen"
pen = turtle.Turtle()
pen.color("blue")      # Set the pen color
pen.pensize(3)         # Set the pen size

# Draw a square
for _ in range(4):
    pen.forward(100)   # Move forward by 100 units
    pen.right(90)      # Turn right by 90 degrees

# Finish drawing
screen.exitonclick()

Enhancements (Optional)

Feel free to experiment with the following to make your square more colorful and interesting:

  • Change Pen Color: 
    pen.color("green")
  • Change Pen Size: 
    pen.pensize(5)

Expected Outcome

When you run the program, a window titled "Challenge 1: Draw a Square" will appear. A blue square with each side 100 units long will be drawn on a light yellow background. The window will remain open until you click on it.
