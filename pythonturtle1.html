Python Turtle Assignment: Challenge 1

Welcome to your first Python Turtle challenge! Follow the steps below to create your first Turtle drawing. Click "Start Challenge" to begin.

Step 1: Import the Turtle Module

The Turtle module is a standard Python library used to create simple graphics. It comes pre-installed, so you don't need to install anything extra.

To use it, you first need to import it into your Python script.

import turtle

Action: Open your code editor and type the above line to import the Turtle module.

Step 2: Create a Turtle Object

A Turtle object acts as your pen that draws on the screen. You'll also create a screen where the turtle will draw.

import turtle

# Create a screen
screen = turtle.Screen()

# Create a turtle named "pen"
pen = turtle.Turtle()

Action: In your Python script, add the above lines to create the screen and turtle objects.

Step 3: Move the Turtle Forward

Let's make the turtle move forward by 100 units. This action will draw a straight line.

pen.forward(100)

Action: Add the above line to your script to move the turtle forward.

Step 4: Turn the Turtle Right

To draw a square, the turtle needs to turn 90 degrees to the right after drawing each side.

pen.right(90)

Action: Add the above line to turn the turtle right by 90 degrees.

Step 5: Repeat Moving and Turning

To complete the square, repeat the move and turn steps three more times.

# Draw first side
pen.forward(100)
pen.right(90)

# Draw second side
pen.forward(100)
pen.right(90)

# Draw third side
pen.forward(100)
pen.right(90)

# Draw fourth side
pen.forward(100)
pen.right(90)

Action: Add the above lines to your script to draw all four sides of the square.

Step 6: Add a Loop to Simplify

Instead of writing the same code four times, you can use a loop to make the code cleaner and more efficient.

for _ in range(4):
    pen.forward(100)
    pen.right(90)

Action: Replace the repetitive code with the above loop to draw the square.

Step 7: Keep the Window Open

After drawing, you want the window to stay open until you click on it. This allows you to view your drawing without the window closing immediately.

screen.exitonclick()

Action: Add the above line at the end of your script to keep the window open.

Complete Code

Here is the complete code for Challenge 1. You can copy and paste it into your Python script.

import turtle

# Set up the screen
screen = turtle.Screen()
screen.title("Challenge 1: Draw a Square")
screen.bgcolor("lightyellow")  # Optional: Change background color

# Create a turtle named "pen"
pen = turtle.Turtle()
pen.color("blue")      # Set the pen color
pen.pensize(3)         # Set the pen size

# Draw a square
for _ in range(4):
    pen.forward(100)   # Move forward by 100 units
    pen.right(90)      # Turn right by 90 degrees

# Finish drawing
screen.exitonclick()

Enhancements (Optional)

Feel free to experiment with the following to make your square more colorful and interesting:

  • Change Pen Color: 
    pen.color("green")
  • Change Pen Size: 
    pen.pensize(5)
  • Change Background Color: 
    screen.bgcolor("lightblue")
  • Change Turtle Speed: 
    pen.speed(2)

Expected Outcome

When you run the program, a window titled "Challenge 1: Draw a Square" will appear. A blue square with each side 100 units long will be drawn on a light yellow background. The window will remain open until you click on it.
